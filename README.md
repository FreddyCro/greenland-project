# greenland-project

## Project setup

```bash
npm install
```

### Setup env

1. copy .env.example => .env
2. 依照以下說明修改

```env
# 產品模式
VUE_APP_MODE=test/production

# 測試環境專案根目錄
VUE_APP_TEST_PATH=

# 正式環境專案根目錄
VUE_APP_PRODUCTION_PATH=

# 影片素材根目錄
VUE_APP_VIDEO_PATH=
```

### Compiles and hot-reloads for development

npm run serve
